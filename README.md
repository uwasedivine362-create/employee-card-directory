This assignment is about building a React application that fetches employee data from a public API and displays it in a clean and professional way.

You will use React Hooks like useState and useEffect to manage data, loading, and errors. The app will show employee information (name, email, phone, company, etc.) using reusable components called EmployeeCard.

The application must also include a search feature that filters employees by name or email in real time. All data should be displayed dynamically using JavaScript methods like .map().

Additionally, the app should have a responsive design (works on mobile, tablet, and desktop) and may include extra features like printing or downloading employee cards.

Overall, the goal is to practice API fetching, component structure, dynamic rendering, and modern React
